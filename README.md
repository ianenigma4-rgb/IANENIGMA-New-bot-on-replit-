## 🚀 Getting Started

### Requirements

- Node.js v18 or higher
- A WhatsApp account
- A session ID (see below)

### Installation

```bash
git clone https://github.com/ianenigma4-rgb/PATRON-MD3
cd PATRON-MD3
npm install
```

### Configuration

Edit the `.env` file with your details:

```env
OWNER_NUMBER=your_number_here
BOT_NAME=IAN ENIGMA MD
OWNER_NAME=Ian Enigma
SESSION_ID=your_session_id_here
PREFIX=.
```

### Run the Bot

```bash
node index.js
```
