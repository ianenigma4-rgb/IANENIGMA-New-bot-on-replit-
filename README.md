## 🚀 Deploy to Hosting Panel (Katabump / any Node.js panel)

### Required Environment Variables

| Variable | Required | Description |
|----------|----------|-------------|
| `PHONE_NUMBER` | **Yes** | Your WhatsApp number without + (e.g. `256775063416`) |
| `OWNER_NUMBER` | No | Owner number for sudo access |
| `PORT` | No | Keep-alive server port (default: `3000`) |

### Deploy Steps

1. **Import** this repo into your hosting panel
2. **Set Environment Variable**: `PHONE_NUMBER=256775063416`
3. **Build Command**: `npm install --legacy-peer-deps`
4. **Start Command**: `node index.js`
5. **Start the bot**
6. Pairing code appears in logs — enter it in WhatsApp: **Settings > Linked Devices > Link with phone number**

## ⚙️ Features

- ✅ **Pairing code login** — no QR scan needed on hosting panels
- ✅ **Auto-reconnect** with conflict handling for 24/7 uptime
- ✅ **Batman/Gotham themed** UI
- ✅ **100+ commands**: downloads, group admin, AI chat, games, fun
- ✅ **Express keep-alive server** for uptime monitoring
- ✅ **Lightweight memory management** with auto-GC
- **Tag all group members** with `.tagall`
- **Admin restricted usage** for sensitive commands
- **Games**: Tic-Tac-Toe, Hangman, Trivia
- **Media**: Sticker, blur, screenshot, background removal
- **Downloads**: YouTube, TikTok, Instagram, Spotify, Facebook
- **AI Chatbot** with `.chatbot` and `.ai`
- **Anti-link & Anti-tag** protection for groups

## 📖 About

The Knight WhatsApp Bot assists group admins by providing them with tools to efficiently manage large WhatsApp groups. The bot uses the Baileys library to interact with the WhatsApp Web API and supports multi-device features.

It is lightweight and can be easily customized to add more commands as per your requirements. The bot runs in a Node.js environment and provides QR code-based authentication to link your WhatsApp account.

- Node.js installed on your system
